# Cedrus Dang Portfolio

Personal portfolio website showcasing projects, skills, and experience in **data, analytics, and software systems**.

**Live Demo:** [https://www.cedrusdang.com](https://www.cedrusdang.com)

## About

* **Author:** Cedrus Dang
* **Background:** Data and software systems
* **Focus:** Data processing, analytical workflows, and system-oriented problem solving

## Features

* Project gallery covering data, analytics, and system-focused work
* Skills overview across data, engineering, and software development
* Professional background and project context
* Responsive web design
* Contact information

## Tech Stack

* React (Create React App)
* Material UI
* JavaScript, HTML, CSS
* GitHub Pages

## Getting Started

1. Install dependencies

   ```bash
   npm install
   ```

2. Run local development server

   ```bash
   npm start
   ```

3. Build for production

   ```bash
   npm run build
   ```

4. Deploy to GitHub Pages

   ```bash
   npm run deploy
   ```
